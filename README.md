# MajdataPlay-Skins

## 提交

请确保您要提交的皮肤

- 无侵权风险
- 可以在MajdataPlay最新版正常游玩

选中您要提交的皮肤的全部文件,
> [!NOTE] 请这样做
> ![alt text](./.github/resources/right-zipped-files.png)

而不是选中单个皮肤文件夹, 
> [!NOTE] 请不要这样做
> ![alt text](./.github/resources/wrong-zipped-files.png)

右键->发送到->压缩(zipped)文件夹, 随后将生成的.zip文件的名字更改为你的皮肤名字
